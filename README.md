# Tag Extractor
A Java Swing application that extracts keywords from text files by analyzing word frequency and filtering out common stop words.

## Features
- Load and analyze text files
- Filter stop words outlined in a file you choose
- Display tags/keywords sorted by frequency
- Save extracted tags to file

## How to Run
1. Run `TagExtractorRunner.java`
2. Load a text file (e.g., from Project Gutenberg)
3. Load stop words file
4. Click "Extract Tags"
5. Save results
